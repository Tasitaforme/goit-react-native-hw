

# [React Native](https://reactnative.dev/)

Щоб створити проект з назвою "AwesomeProject", перейти у папку проекту і запустити його, потрібно прописати у терміналі команди:

```
npx create-expo-app AwesomeProject
cd AwesomeProject
npx expo start
```
